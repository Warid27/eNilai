# eNilai — Website Input Nilai Siswa

eNilai adalah aplikasi berbasis web sederhana untuk mengelola input nilai siswa. Website ini dibuat sebagai tugas sekolah, menggunakan **PHP Native** tanpa framework.

## ✨ Fitur Utama

- ✅ **Login Sistem**  
  Otentikasi pengguna berdasarkan username dan password.

- 📊 **Dashboard**  
  Menampilkan ringkasan data dan navigasi cepat ke fitur utama.

- 🧑‍💼 **Level/Role Pengguna**  
  - Admin: Kelola semua data
  - Guru: Input dan edit nilai siswa
  - Viewer: Hanya bisa melihat data

- ✏️ **CRUD Nilai Siswa**  
  - **Create**: Tambah nilai baru  
  - **Read**: Lihat daftar nilai  
  - **Update**: Edit nilai yang sudah ada  
  - **Delete**: Hapus nilai

## 🛠️ Teknologi

- **Backend**: PHP Native
- **Database**: MySQL (phpMyAdmin)
- **Frontend**: HTML, CSS, JavaScript (optional)

## 🚀 Cara Menjalankan

1. Clone atau download repository ini
2. Letakkan di folder `htdocs` (jika menggunakan XAMPP)
3. Import file `database.sql` ke phpMyAdmin
4. Ubah konfigurasi database di `config/db.php`
5. Jalankan di browser via `http://localhost/enilai`
